Projects

A collection of personal and academic projects showcasing my skills and passion for development.

Webscraper — Price Comparison Tool

Developed a web application that automatically scrapes product prices from multiple e-commerce platforms including Amazon, eBay, Myntra, and Snapdeal. Users can search for a product, filter results by price range, and view the best available deals across different websites, all in one interface.

Technologies: Python, Flask, BeautifulSoup, HTML, CSS

Key Features:

• Real-time price comparison across multiple online stores

• Filter products by price range

• User-friendly web interface for easy navigation

• Error handling for unavailable products or network issues

GitHub : Hemavarnakrish/Webscraper-PriceComparison.git

Tech Escape Room — Interactive Puzzle Game

Created a browser-based interactive escape room game for college events. The game consists of two themed rooms where participants solve puzzles, riddles, and logical challenges to advance to the next level. Each room includes hints and a scoring system to enhance engagement and encourage problem-solving.

Technologies: HTML, CSS, JavaScript

Key Features:

• Two unique escape rooms with progressive challenges

• Interactive puzzle system with hints for incorrect answers

• Score tracking and time-based challenges for competition

GitHub : Hemavarnakrish/TechEscapeRoom.git
